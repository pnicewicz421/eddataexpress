About This Archive

The ED Data Express Archive project was created to preserve access to important educational data published by the U.S. Department of Education. This archive includes:

  • Complete snapshots of the website HTML, CSS, and JavaScript
  • Extracted data tables in both CSV and Parquet formats
  • Archived images, videos, documents, and other media
  • A query interface for exploring and analyzing the data
Getting Started

Here are some ways to get started with the ED Data Express Archive:

  1. Browse the archived website to explore pages as they appeared
  2. Use the Data Explorer to query and analyze education datasets
  3. Access media files including images and documents

For developers: All data is also available through our JSON API endpoints.
